# ArgoMobile - Cross-Platform Mobile App for ARGODREIGN

This mobile app replicates the web dashboards for ARGODREIGN IoT Alert Engine using Expo (React Native).

## Setup
1. Install Expo CLI: `npm install -g @expo/cli`
2. Install dependencies: `npm install`
3. Start development server: `npx expo start`

## Running
- Scan QR code with Expo Go app on phone
- Or run on simulator/emulator

## Features
- Navigation between all 7 dashboards
- API integration with Spring Boot backend (placeholder)
- Cross-platform (Android/iOS)
